Word Lists

Public Lists​

User Lists​

A user list is a word list created and managed by a WotB user. Users must be logged in to interact with their lists.

Create a List​

To create a list, navigate to YOUR LISTS and click the New List button.

Fill out the form with the following fields:

  • Name: The name of your list.
  • Description: (Optional) A short description of your list.
  • Lexicon: The list is linked to a specific Scrabble lexicon. Words added to the list will be validated against this lexicon.
  • Order: Defines the order in which words will be presented during [Regular Practice]. This can be updated later in [List Settings].
    • Random: Words will be chosen randomly, but only from those not seen yet. This ensures no word is repeated until all others have been reviewed.
    • Probability (Descending): Words will be presented in descending order of likelihood to draw.
  • Add File with Words: (Optional) Upload a file containing one word per line. Words should be in their canonical form (lemma), e.g., "mouse" instead of "mice." You can also add words manually later.
  • Look for Duplicates on All My Lists: Prevent adding words that already exist in your other lists by enabling this checkbox.

Good to know:

  • Due to mobile frontend requirements, only words with 8 letters or fewer can currently be added. However, a canonical form longer than 8 letters is allowed if it has a valid shorter form.

Manage a List​

Delete a List​

Deleting a list is irreversible ("hard delete"), meaning all metadata related to the list will be permanently lost. Use with caution.

Add Words​

Click the Add Word button to open a popup. You can add words using one of two methods:

  • Add Single Word: Add a single word with an optional note.
  • Add from List: Add multiple words from a file. For details, see the Create a List section.

Sessions​

View all your past practice sessions in this section.

Settings​

Modify your list's metadata (name, description, drawing order) and adjust additional parameters:

  • (SR) New Cards Daily Limit: Set a daily limit for new cards in spaced repetition practice.
  • Show Definition During Practice: Enable this to display word definitions during practice.
  • Show Note During Practice: Enable this to display your custom notes during practice.
  • Show Extensions During Practice: Enable this to display valid word extensions during practice.
